# Task Item Images

Place task item images here as JPG files. The app will load these files first, and fall back to the built-in placeholder if a file is missing.

Use these exact filenames:

- `mosquito_net.jpg`
- `petrol.jpg`
- `water_container.jpg`
- `shaving_mirror.jpg`
- `sextant.jpg`
- `emergency_rations.jpg`
- `sea_chart.jpg`
- `floating_cushion.jpg`
- `rope.jpg`
- `chocolate_bars.jpg`
- `waterproof_sheet.jpg`
- `fishing_rod.jpg`
- `shark_repellent.jpg`
- `rum.jpg`
- `vhf_radio.jpg`

Recommended shape: landscape, around 16:11 or 4:3. Larger files are fine, but keeping each image under 500 KB will make the page feel faster.
