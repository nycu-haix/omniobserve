# 暗樁參與者發言提綱：Lost at Sea

本文件僅供研究團隊內部準備使用，請勿提供給一般參與者。

## 1. 角色設定

暗樁在討論中扮演一般參與者。他的發言風格應該是：

- 有自己的判斷邏輯。
- 願意聽其他人的理由。
- 會提出不同看法，但不壓過所有人。
- 說話自然，不要像在背稿。
- 不提研究目的、系統提示、實驗條件或自己與研究團隊的關係。

暗樁的整體立場可以集中在這個方向：

> 我會比較重視定位、資訊、方向感和長期生存，因為我們不確定救援什麼時候來，所以不能只把希望放在短時間內被看到。

## 2. 暗樁初始排序建議

暗樁可在個人思考階段先使用以下排序。若小組討論時其他人提出有力理由，可以做小幅調整。

| 排名 | 物品 |
| ---: | --- |
| 1 | 離線 GPS 定位器 |
| 2 | 大西洋紙本地圖 |
| 3 | 小型收音機 |
| 4 | 釣魚竿 |
| 5 | 繩子 |
| 6 | 蚊帳 |
| 7 | 裝水容器 |
| 8 | 緊急糧食 |
| 9 | 防水塑膠布 |
| 10 | 巧克力棒 |
| 11 | 可漂浮的坐墊 |
| 12 | 防鯊噴劑 |
| 13 | 刮鬍鏡／小鏡子 |
| 14 | 一罐汽油 |
| 15 | 萊姆酒／高濃度消毒酒精 |

若系統當天顯示的是「高濃度消毒酒精」而不是「萊姆酒」，暗樁直接稱呼為「酒精」即可。

## 3. 個人思考階段可輸入或說出的內容

5 分鐘個人思考階段中，暗樁可以留下以下 private thoughts。內容不用全部照念，可選幾句自然地說。

1. 「我覺得 GPS 和地圖應該放前面，至少要先知道自己在哪裡，不然等救援有點太被動。」
2. 「收音機雖然不能發訊息，但可以接收資訊，也許可以知道天氣或附近救援狀況。」
3. 「鏡子要有太陽、角度也要對，我不確定它能不能穩定派上用場。」
4. 「汽油在救生艇旁邊使用感覺有風險，而且用掉就沒了。」
5. 「蚊帳不一定只能防蚊，也許可以遮陽、當簡單網子或過濾一些雜物。」
6. 「釣魚竿對長時間漂流可能很重要，因為現有食物總會吃完。」

## 4. 公開討論發言流程

以下內容是暗樁可以在 15 分鐘討論中依照情況使用的發言。不要逐字全部講完，根據現場討論自然插入即可。

### 第 0 到 2 分鐘：先說明自己的排序邏輯

> 我先講一下我的排序邏輯。我覺得我們不應該完全假設一定會很快有人來救。題目說我們離陸地數百英里，而且通訊壞掉，所以我會把能知道位置、判斷方向、增加長期存活機會的東西放前面。
>
> 所以我目前會把 GPS、地圖、收音機放很前面。GPS 至少知道座標，地圖可以知道大方向，收音機可能可以聽天氣或救援相關資訊。

如果有人說收音機不能發訊號：

> 對，它不能發訊號，但至少能接收資訊。完全沒有資訊的話，我覺得大家只能瞎等。就算只能知道天氣或附近狀況，也比完全沒有資訊好。

### 第 2 到 5 分鐘：談小鏡子與汽油

> 我知道鏡子可以反光，但我會擔心它太吃條件。要有太陽、角度要對、也要剛好有人在可視範圍內。晚上或陰天就比較難用。
>
> 汽油我也有點猶豫。救生艇旁邊用火其實有風險，而且汽油只有一罐，用掉就沒了。我會比較傾向把它當最後手段。

如果其他人很支持小鏡子：

> 我同意如果剛好有船或飛機經過，鏡子會很有用。我只是擔心它太依賴時機，所以我會想把一些比較穩定的工具放前面。

如果其他人很支持汽油：

> 我同意它可能可以製造很明顯的訊號，但我會擔心安全性和使用時機。可能要等真的看到救援才用，不一定要排最前面。

### 第 5 到 8 分鐘：推長期生存工具

> 釣魚竿我會排滿前面，因為如果漂很多天，食物一定會是問題。緊急糧食和巧克力有限，但釣魚竿至少有機會持續取得食物。
>
> 繩子也應該滿前面，因為很多東西都要固定，不管是固定人、物品、塑膠布，或是做一些簡單結構。

如果有人說水比食物重要：

> 水當然重要，所以我沒有把水排很後面。但如果只有一個裝水容器，我們還是需要想辦法收集雨水。我的意思是，長期存活不能只靠現有物資。

### 第 8 到 11 分鐘：談蚊帳、地圖與 GPS

> 蚊帳我覺得可能沒有大家想得那麼沒用。它可以遮陽，也可能拿來當簡單網子或過濾雜物。不是最重要，但我不會放到最後。
>
> 地圖我還是覺得重要。GPS 如果給座標但沒有地圖，其實座標也很難理解；地圖和 GPS 應該一起看。即使不能精準航行，至少知道哪個方向比較可能有陸地或航線。

如果其他人說救生艇無法導航：

> 我不是說我們一定能划到陸地，而是至少不要完全沒有方向。洋流或風也可能讓我們移動，如果知道大方向，決策會比較有根據。

### 第 11 到 13 分鐘：嘗試整理前五名

> 我覺得我們可以先不要糾結每個細節，先決定前五個的大方向。我的想法是前五偏「定位、資訊、長期生存」，所以會放 GPS、地圖、收音機、釣魚竿、繩子。
>
> 如果你們覺得水一定要前五，我可以接受把水換進來，但我會希望 GPS 和地圖至少不要太後面。

### 第 13 到 15 分鐘：接近收斂時的說法

> 快到時間的話，我建議我們先定一個方向：不要只靠被動等待。我會把定位和資訊來源放前面，然後是食物、水、工具。鏡子和汽油我覺得可以留著，但不要排太前面，因為它們都比較看時機。
>
> 我可以接受最後排序不是完全照我的，但我真的覺得 GPS、地圖、收音機不應該太後面。

## 5. 常見反駁應對句

### 若有人強烈支持小鏡子

> 我懂反光求救的邏輯，但我擔心它太依賴外部條件。沒有太陽、距離太遠、方向不對，效果就會受影響。

### 若有人強烈支持汽油

> 汽油的訊號效果可能很強，但它也有風險。救生艇空間小，旁邊又都是人，我會比較謹慎使用。

### 若有人說 GPS 沒用

> GPS 至少提供確定的位置資訊。就算不能通訊，我們後面做任何判斷都需要知道自己在哪裡，不然完全沒有基準。

### 若有人說地圖沒用

> 單獨看地圖可能沒用，但搭配 GPS 就有用。沒有地圖，GPS 座標也只是數字。

### 若有人說蚊帳在海上沒用

> 我同意不是拿來防蚊，但材料本身還是可用，像遮陽、簡單網子、過濾雜物。我不會排前五，但也不會放最後。

### 若有人說收音機不能發送訊號

> 對，它不能呼救。不過能接收資訊還是比完全沒有資訊好，尤其是天氣或附近救援活動。

### 若有人說應該以求救訊號為主

> 我同意要能被救，但我覺得不能把希望全部壓在剛好有人看見我們。萬一短時間沒人經過，定位、食物和工具會更重要。

## 6. 說話風格提醒

暗樁應該：

- 像一般參與者一樣自然討論。
- 偶爾詢問其他人的排序或理由。
- 在小地方讓步，讓討論可以繼續。
- 持續用「定位、資訊、長期生存」這個邏輯解釋自己的排序。
- 不要說自己知道答案。
- 不要說太多，保留其他人發言空間。

暗樁不應該：

- 攻擊或嘲笑其他參與者。
- 說明顯不合理的話，例如「鏡子不能反光」。
- 假裝自己有專業航海知識。
- 在小組已明確拒絕某論點後仍一直重複。
- 暴露自己是研究團隊成員。
- 提到 Similarity Cue、backchannel 或研究假設。
