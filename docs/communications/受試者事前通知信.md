# 受試者事前通知信

本文件只保留一封預計寄出的 Mailmeter Batch Mail 正式通知信。信件可提到實驗時間、地點、設備、個別 Meet、系統連結與 Canva 連結；不得透露 task order、cue/no-cue 條件、暗樁安排或研究假設。

## Batch Mail 欄位

CSV 欄位名稱需和以下 `{{}}` 內文字一致：

- `受試者姓名`
- `實驗日期`
- `實驗時間`
- `報到時間`
- `實驗地點`
- `Google_Maps_連結`
- `個別_Meet_連結`
- `系統連結_1`
- `系統連結_2`
- `個人_Canva_連結`
- `小組_Canva_連結`
- `聯絡人姓名`
- `聯絡人Email`

## 信件主旨

```text
[OmniObserve] 感謝參與實驗 - 請於 {{實驗日期}} {{實驗時間}} 出席
```

## 信件正文

```text

以下為本次實驗的相關資訊：
```

## 寄出前檢查

- [ ] 每位受試者都有自己的 `{{個別_Meet_連結}}`，且不是全組共用討論連結。
- [ ] `{{系統連結_1}}` 與 `{{系統連結_2}}` 已依該受試者的 task 順序填入正確連結。
- [ ] 每位受試者的 `{{個人_Canva_連結}}` 是獨立給該受試者的 Canva 連結。
- [ ] 同一組受試者的 `{{小組_Canva_連結}}` 填入同一份小組共用 Canva 連結。
- [ ] 所有系統與 Canva 連結都有提醒「會議當下依研究人員指示再開啟」。
- [ ] 信件沒有提到 task order、cue/no-cue 條件、暗樁安排或研究假設。
